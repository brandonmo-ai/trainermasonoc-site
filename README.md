# Asset Guide

Add final production images here using these exact paths:

- `assets/mason-hero.jpg`
- `assets/mason-about.jpg`
- `assets/transformations/transformation-1.png`
- `assets/transformations/transformation-2.png`
- `assets/transformations/transformation-3.png`

Recommended notes:

- Keep portrait images high resolution.
- Transformation images should stay vertically oriented for the existing layout.
- The site already supports `width: 100%`, `height: 100%`, `object-fit: contain`, and `object-position: center top` in the transformation section.
